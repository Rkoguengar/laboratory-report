# Цель работы

## Основная цель

Приобрести практические навыки установки операционной системы на виртуальную машину и настройки минимально необходимых сервисов для дальнейшей работы.

# Ход выполнения работы

## Создание виртуальной машины в VirtualBox

- Среда виртуализации: **Oracle VirtualBox**
- Имя ВМ: **rishardkogengar**
- ISO-образ: **Rocky-10.1-x86_64-dvd1.iso**
- Тип ОС: **Linux → Red Hat → Red Hat (64-bit)**
- Автоустановка отключена (ручная настройка параметров)

## Базовые параметры виртуальной машины

- ОЗУ: **4096 МБ**
- CPU: **4 виртуальных процессора**
